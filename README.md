# Сегментация и когортный анализ клиентов (SQL)

Четыре аналитических запроса на основе таблиц транзакций и анкетных данных клиентов
за период 06.2015–06.2016.

## Задачи и решения

**1. Лояльные клиенты**
Клиенты с покупками в каждый месяц года подряд (`HAVING COUNT(DISTINCT month) >= 12`),
с расчётом среднего чека и количества операций.

**2. Помесячная динамика**
CTE с агрегатами по месяцам + доля каждого месяца от годовой выручки и числа операций
(сравнение с `year_totals`).

**3. Распределение трат по полу**
Window function `SUM(...) OVER (PARTITION BY month)` — доля затрат каждого пола
в общей выручке месяца, без дополнительных подзапросов.

**4. Возрастные когорты**
Сегментация на группы по 10 лет + разбивка среднего чека по кварталам
через `CASE WHEN ... END` внутри `AVG()`.

## Использованные техники
CTE, multi-table JOIN, window functions, conditional aggregation, HAVING с DISTINCT.
